Z64C supports a wide range of commands orientated at playing and analyzing chess
games.

boardprint
	Prints the current chess position on the screen.
debug [true | false]
	Sets the debug setting.
go
	Starts analyzing the current chess position with the given settings.
help
	Print a list of available UCI commands.
islegal
	Test the legality of a move. You must write the move in coordinates
	notation.
isready

move
	Make a move and update the chess position accordingly.
ponderhit
position
	Use this command to change the current chess position to any.
quit
	Exit this program.
stop
	Stop analyzing the game.
ucinewgame
	Start a new game and delete all data analysis about the current one.
